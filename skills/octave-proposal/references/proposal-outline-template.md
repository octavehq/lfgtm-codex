# Proposal Outline Template

Present this to the user for approval before generating. Show the six jobs (stage-trimmed if relevant), what each will contain for *this* account, and the Octave sources backing it. Keep each line concrete — no placeholder filler.

```
PROPOSAL OUTLINE: [Headline]
=============================

Prepared for: [Company]
Champion: [Name, Title — or "general executive audience"]
Stage: [Deal stage]
Key concerns: [Listed concerns]
Pricing: [Yes / No / TBD]

---

THE SIX JOBS
------------

1. THE BOTTOM LINE
   - Situation + gap: [one line]
   - What we propose: [one line]
   - Expected outcome: [one line]
   - Takeaway: [the one thing to remember]

2. THE PROBLEM WE'RE SOLVING
   - [Challenge 1 — grounded in call finding / their words]
   - [Challenge 2 — from enrichment or conversation]
   - Cost of inaction: [one line]

3. WHAT WE'D DO
   - [Capability 1 → Solves for: pain 1]
   - [Capability 2 → Solves for: pain 2]
   - [Capability 3 → Solves for: pain 3]
   - Per-persona framing (from Motion ICP cell):
     - For [persona A]: [what it means for them]
     - For [persona B]: [what it means for them]

4. WHY IT'S WORTH IT
   - Business case: [value vs. cost of inaction]
   - ROI framing: [approach — qualitative / their numbers / labeled model]
   - Competitive angle: [only if competitor in deal — else omit]
   - Investment: [pricing table / TBD / omitted]

5. PROOF IT WORKS
   - [Result 1 — same industry/size, with metric]
   - [Result 2 — similar use case, with metric]
   - Logos: [recognizable customers, if available]

6. HOW IT GOES
   - Timeline: [phases, time-to-first-value]
   - Needed from them: [one line]
   - Next step: [one concrete action]

---

Stage trim applied: [none / dropped Job X / led with Jobs 4+6 / renewal reframe]

Octave sources:
- Company profile: [Company] — [key insights]
- Motion ICP cell: [persona × segment] — [per-persona angles]
- Proof points: [N] references pulled — [which companies]
- Competitive intel: [competitor + status, or "none in deal"]
- Findings: [N] call findings — [what was actually said]

---

Does this outline look good? I can:
1. Proceed to brand resolution and generation
2. Add / remove / reorder jobs
3. Go deeper on any job
4. Change the narrative angle
```
